Django-Redux Example
=============================

Minimalistic example app that uses django-redux to dispatch and responds to React/Redux actions.

Quickstart
----------

::

    $ npm install
    $ npm install --devOnly
    $ npm run compile

    $ pip install -r REQUIREMENTS.txt
    $ ./manage.py migrate
    $ ./manage.py runserver
